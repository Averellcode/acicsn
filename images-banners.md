# Guide des Images de Bannière pour ACIC-SN

Ce document liste les suggestions d'images de bannière adaptées à chaque page du site ACIC-SN.

## Sources d'images gratuites recommandées

- **Unsplash** : https://unsplash.com (Licence libre, haute qualité)
- **Pexels** : https://www.pexels.com (Licence libre, haute qualité)
- **Pixabay** : https://pixabay.com (Licence libre, grande variété)

## Dimensions recommandées

- **Largeur** : 1920px (pour écrans larges)
- **Hauteur** : 600-800px (pour les hero sections)
- **Format** : JPG ou WebP (optimisé pour le web)
- **Poids** : Maximum 500KB (pour un chargement rapide)

## Instructions d'intégration

1. **Télécharger les images** depuis les sources recommandées
2. **Optimiser les images** :
   - Utiliser un outil comme TinyPNG ou ImageOptim
   - Réduire le poids à moins de 500KB
   - Convertir en WebP si possible
3. **Placer les images** dans le dossier `images/banners/`
4. **Nommer les fichiers** :
   - `hero-home.jpg` (page d'accueil)
   - `banner-a-propos.jpg`
   - `banner-produits.jpg`
   - `banner-domaines.jpg`
   - `banner-services.jpg`
   - `banner-marques.jpg`
   - `banner-actualites.jpg`
   - `banner-agence.jpg`
   - `banner-contact.jpg`
   - `banner-devis.jpg`

## Exemple d'utilisation dans le CSS

```css
.hero {
    background-image: url('../images/banners/hero-home.jpg');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
}

.page-header {
    background-image: url('../images/banners/banner-[page].jpg');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
}
```

## Notes importantes

- Toutes les images doivent respecter la charte graphique ACIC-SN (tons bleus, professionnel)
- Ajouter un overlay sombre si nécessaire pour améliorer la lisibilité du texte
- Tester sur différents appareils pour s'assurer de la qualité d'affichage
- Vérifier les droits d'utilisation des images (toutes les sources mentionnées sont libres de droits)
